## Overview

This portfolio site serves as both a professional academic presence and a technical demonstration. Built with Next.js 14 and the App Router, it reads content from MDX files at build time, producing a fast, statically generated site with zero hosting costs.

## Key Features

- Progressive Web App (PWA) — installable on phones and desktops
- MDX-based content system — add new research or projects by creating a file
- Responsive design with dark theme
- Animated page transitions and scroll-triggered reveals
- Canvas-based particle field animation on the homepage
