## Overview

This project develops a graphical user interface (GUI) built on FEM principles that connects to a rigorous Fortran F90-based SPH (Smoothed Particle Hydrodynamics) solver engine. The primary goal is to make SPH simulation accessible through intuitive GUI design while maintaining computational rigor in the backend solver.

## Key Features

- **FEM-Based GUI**: Intuitive graphical interface for pre-processing, mesh generation, and problem setup
- **Fortran F90 SPH Engine**: High-performance Smoothed Particle Hydrodynamics solver written in Fortran for computational efficiency
- **Large Deformation Focus**: Designed specifically for problems involving extreme material deformation where traditional FEM fails
- **Software Implementation**: Emphasis on usability and practical deployment for geotechnical engineering applications

## Technical Approach

- GUI handles geometry definition, boundary conditions, and material property input
- SPH kernel manages particle-based discretization and time integration
- Post-processing visualization of displacement, velocity, and stress fields
- Validation against analytical solutions and benchmark problems
